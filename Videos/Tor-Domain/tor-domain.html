How to make a FM radio station using a raspberry pi

Github repo: https://github.com/cathugger/mkp224o

Reasons to Do this

To be able to have a custom domain, to help verify the owner, less to remember

How to get it

So first you want to run the following in a terminal to install the right library for it

sudo apt install git gcc libc6-dev libsodium-dev make autoconf

Then you want to you download the source code from github using the website or using git like below.

git clone https://github.com/cathugger/mkp224o && cd mkp224o

After installing and going into the directory run the following to setup the right configuration files and to compile the whole thing

./autogen && ./configure && make

When thats done installing you are now able to make your own tor domains. Remember to put then in /var/lib/tor/hidden_service if you want to use it.
